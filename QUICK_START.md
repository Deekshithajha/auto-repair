## ğŸ”§ Troubleshooting

### Port Already in Use?
```bash
# Kill process on port 3000
npx kill-port 3000

# Or use a different port
npm run dev -- --port 3001
```

### Dependencies Not Installing?
```bash
# Clear cache and reinstall
```

```bash
# Check Node version (need 16+)
node --version
```
